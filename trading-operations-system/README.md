# Sistema de Gestión de Operaciones de Trading

Aplicación de escritorio desarrollada en Python para gestionar y registrar operaciones financieras.

## Descripción

El sistema permite registrar, validar y analizar operaciones de trading, asegurando consistencia y trazabilidad en los datos.

## Funcionalidades principales

- Registro, modificación y eliminación de operaciones  
- Cálculo automático de resultados (ganancia/pérdida)  
- Resúmenes mensuales y seguimiento de rendimiento  
- Exportación de datos a CSV  
- Funcionalidad de auditoría  
- Validación de datos para prevenir inconsistencias  

## Arquitectura

La aplicación está estructurada bajo el patrón MVC (Model-View-Controller):

- **Modelo**: lógica de datos y estructura  
- **Vista**: interfaz gráfica desarrollada con Tkinter  
- **Controlador**: conexión entre la lógica y la interfaz  

## Tecnologías utilizadas

- Python  
- SQLite  
- Tkinter  

## Objetivo

El proyecto busca mejorar el control, la validación y el seguimiento de operaciones en un entorno estructurado.
